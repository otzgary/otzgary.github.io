自从 AI Coding 工具成熟起来之后，X 上便充斥着巨量的教你如何在本地搭建笔记软件的方法，比如让你用 Obsidian + Claude Code 或者 Cursor 搭建文件系统，Markdown 也被莫名其妙推上神坛……
但我要告诉你：这一切都是没有用的
你回想一下，自己是不是用了很多笔记软件？
从 Evernote（印象笔记） 到 Notion ，从 Flomo 到 Obsidian，市面上有各种各样天花乱坠的笔记软件，但你没有一个能坚持使用，都是交了年费就开始吃灰
你学了一堆记笔记的方法论，弄了一堆漂亮的笔记模板，也是吃灰
原因很简单：就是因为你用这些笔记软件没有反馈 —— 没有反馈就坚持不下去，就这么简单
结论跃然纸上：X 就是最好的笔记软件，没有之一
我个人在使用 X 的时候，甚至从来没有使用过它的草稿箱功能
草稿箱本质上也是一个像 Notion、Obsidian、Evernote 一样的笔记软件，也是没有用的 —— 只有发出去的笔记才是有用的
你肯定要问：我不想公开怎么办？
实际情况是：没啥不可以公开的，你我都没啥国家机密
你又要问：公开出去如果影响了他人对我的看法怎么办？
非常简单：拿个匿名的小号发就行
如果你的笔记有人看，那恭喜你，你的思考是有点牛逼的
如果你的笔记没人看（就像现在一样）那说明可能这个笔记本身也没什么意义
深入一层，让我们来看一看把社交媒体当成笔记软件的本质：在各种社交媒体上发帖本质上是在清理大脑的缓存
在计算机科学中，缓存是为了提高效率而存在的，但如果缓存溢出，系统就会宕机 —— 经常记笔记是有好处的，能防止你的脑子锈掉
更深想一层，这是一种「认知卸载」，你是在给大脑「减负」：
释放工作记忆： 人的大脑就像内存，容量有限。如果你脑子里一直盘旋着某个想法、某种情绪，它会持续占用你的背景带宽。把它写出来、发出去，本质上是将数据从内存转存到了外部硬盘，从而让大脑恢复清爽
情绪代谢： 很多时候，发动态是一种情绪的 Debug。通过文字化，你强迫自己梳理了混乱的逻辑，这种表达本身具有疗愈作用
确认存在感： 就像你经常查查网速、确定没断网，社交媒体的互动能让你确认自己与世界的连接
最后再说一个 Claude Code 出现之后可能会对我们造成的长期伤害：你太容易把自己的各种中间过程、各种草稿直接扔给 AI 让它给你总结了
但AI 总结得越好，你个人的思考能力就越差！
认知科学家提出过一个理论：对于一段信息，你写入的难度越大，提取的难度越小
你想想，把一篇文章转发进收藏夹毫不费力，但根本记不起来
我写过：X 是全世界人思维的游乐场，也是人类上传集体意识的地方
祝你玩得开心！
